# Notes CLI App

A simple terminal-based notes application written in Python.

## What it does

- Add notes from the terminal
- Store notes in a text file (`notes.txt`)
- (Soon) list saved notes

## How to use

### Add a note
```bash
python notes.py add "hello world"
### List notes
python notes.py list

## How it works

Notes are stored line-by-line in a notes.txt file in the same directory.
